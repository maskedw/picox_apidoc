事前条件
  • heap != NULL
  • heap_size > 0
  • block_size > 0
  • alignment >= X_ALIGN_OF(size_t)
  • alignment == 2のべき乗
注意
heapが指すアドレスと、block_sizeはアライメントに切り上げられます。 その結果、意図したブロック分けができていない場合がありますので、はじめからア ラインメントを意識した引数を用意するか初期化後に、必要量を満たせているか確認 した方がよいでしょう。
